# Font Awesome Icons for Clockwork

This directory contains icon fonts for the Clockwork application.

## Download Font Awesome

1. Go to https://fontawesome.com/download
2. Download **Font Awesome 6 Free** (free version)
3. Extract the archive
4. Copy the following file to this directory:
   - `fontawesome-free-6.x.x-desktop/otfs/Font Awesome 6 Free-Solid-900.otf`

5. Rename it to: `fa-solid-900.otf`

## File Structure

```
Assets/Fonts/
├── README.md (this file)
└── fa-solid-900.otf (you need to download this)
```

## Alternative: Direct Download

You can also get the file directly from:
https://github.com/FortAwesome/Font-Awesome/raw/6.x/otfs/Font%20Awesome%206%20Free-Solid-900.otf

**Important**: This font is NOT included in git due to licensing. Each developer must download it separately.
